Introducing EfficientNetB0

Coral Health Assessment Via CNN

This project uses an EfficientNet-B0 CNN to classify reef images into Healthy, Bleached, and Dead coral classes. The canonical benchmark reports 98.11% held-out accuracy, supporting fast and consistent coral health assessment in field conditions.

The Mission

Automating Coral Reef Health Assessment with Explainable AI

This system uses CNN-based image analysis to classify coral reef images into Healthy, Bleached, and Dead categories. It also provides confidence scores and Grad-CAM visual explanations to help users understand predictions.

The objective is to support coral health classification, explainable AI validation, and performance-based assessment for faster reef monitoring decisions.

Technology

Built for Reliable Coral Assessment

EfficientNet-B0 Feature Extractor

Transfer learning with controlled fine-tuning captures reef texture and bleaching patterns while keeping the model compact for deployment.

5-Seed SWA Ensemble Inference

Predictions are averaged across multiple robust checkpoints to improve stability and reduce variance across Healthy, Bleached, and Dead classes.

Grad-CAM and Audit Workflow

Grad-CAM visualizations and per-image audit outputs support transparent behavior checks for field validation and reporting.

Key Findings

  • • EfficientNetB0 Ensemble achieves the highest macro F1 (0.98) with the fewest parameters (20.3M), demonstrating superior parameter efficiency.
  • • ResNet50 Single matches EfficientNetB0's accuracy (98.11%) but requires 16% more parameters and yields a lower macro F1 (0.96).
  • • ConvNeXtTiny Single uses the most parameters (27.8M) yet achieves the lowest accuracy (97.48%), indicating diminishing returns at higher model complexity.
