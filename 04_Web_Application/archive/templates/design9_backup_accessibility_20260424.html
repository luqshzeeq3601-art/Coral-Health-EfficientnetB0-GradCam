Introducing EfficientNetB0

Coral Health Assessment Via CNN

This project uses an EfficientNet-B0 CNN to classify reef images into Healthy, Bleached, and Dead coral classes. The canonical benchmark reports 98.11% held-out accuracy, supporting fast and consistent coral health assessment in field conditions.

Try Model View Metrics

The Mission

Automating Reef Monitoring for Marine Biologists

Manual coral surveys are time-consuming and can vary across observers. This system standardizes assessment by using a trained CNN pipeline to classify coral health from underwater imagery.

Built on an EfficientNet-B0 backbone with transfer learning, the model delivers strong multiclass performance while keeping inference lightweight for practical deployment on diver cameras and edge devices.

Technology

Built for Reliable Coral Assessment

EfficientNet-B0 Feature Extractor

Transfer learning with controlled fine-tuning captures reef texture and bleaching patterns while keeping the model compact for deployment.

5-Seed SWA Ensemble Inference

Predictions are averaged across multiple robust checkpoints to improve stability and reduce variance across Healthy, Bleached, and Dead classes.

Grad-CAM and Audit Workflow

Grad-CAM visualizations and per-image audit outputs support transparent behavior checks for field validation and reporting.
